Collatz Conjecture ("3x + 1" Problem)

"The most dangerous problem in mathematics"

The conjecture states that no matter which positive number you start with, you will always reach 1 eventually.

No number has been found that does not adhere to this pattern.

Not for lack of trying... this problem has been calculated up to 268 (295,147,905,179,352,825,856), that's 200 Quintillion numbers!

Rules:

  • If number x is odd: Multiply by 3 and add 1
  • If number x is even: Divide by 2
